# 支持向量机 support vector machine

SVM的核心思想是

找一条最优的SVM（超平面）

让这条线到两边“最近的样本点”的距离最大（最大间隔超平面）。

支持向量： 离分界线最近的那些点，这些点决定了这条线的位置，如果删除远处的点，线几乎不变，如果移动最近的点，线会变。

间隔越大，模型越稳，泛化能力越强，可以减少过拟合
现实中的意义，最大间隔意味着，如果来了一个新客户，有点稍微波动也不会轻易分类错误。

```python
from sklearn import svm

clf = svm.SVC(kernel='linear')  # 使用线性核函数
clf.fit(X_train, y_train)
y_pred = clf.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
```
